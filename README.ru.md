# Process Launcher

Лёгкий лаунчер для Windows и macOS: добавляй программы один раз и
запускай их в один клик, по горячей клавише или целыми наборами.
Приложение следит за запущенными процессами, умеет перезапускать
упавшие и живёт в системном трее.

*(English version: [README.md](README.md))*

---

## Установка

**Windows.** Скачай `ProcessLauncher-Setup-1.0.exe` со страницы
[релизов](https://github.com/Ruha24/ProcessLauncher/releases),
запусти и следуй мастеру. Установщик создаст ярлык в меню Пуск
(и по желанию на рабочем столе). Права администратора не нужны.

**macOS.** Скачай `ProcessLauncher.dmg`, открой его и перетащи
приложение в папку Applications. При первом запуске, если система
покажет предупреждение о неизвестном разработчике — правый клик
по приложению → «Открыть».

## Как пользоваться

**Добавить программу.** Нажми «+ Add» и выбери .exe, либо просто
перетащи файл (или ярлык .lnk/.url) в окно. Ярлыки распознаются
автоматически — добавится сама программа, а не ярлык.

**Запуск и остановка.** Кнопка «Start» рядом с программой, двойной
клик по строке, или назначь горячую клавишу кнопкой «Bind». Правый
клик по программе открывает меню: перезапуск, авто-перезапуск,
открыть папку, копировать путь, заметка.

**Профили.** Группируй программы по вкладкам (например «Работа» и
«Игры»). Для профиля можно задать цвет и иконку (кнопка «Style»),
горячую клавишу и запускать все программы разом кнопкой «Start all».

**Автозапуск.** Галочка «Start with Windows» в шапке — лаунчер будет
стартовать вместе с системой. В настройках можно выбрать профиль,
который запустится автоматически при открытии лаунчера.

**Слежение за падениями.** Включи «Auto-restart» у программы через
правый клик — если она вылетит, лаунчер поднимет её снова. Все
падения и ошибки пишутся в журнал (кнопка «Log»).

**Менеджер автозагрузки (Windows).** Кнопка «Startup» показывает все
программы, которые стартуют вместе с Windows, и позволяет включать,
выключать и удалять их — как в Диспетчере задач.

**Настройки** (кнопка «Settings»): тёмная/светлая тема, поведение при
закрытии (сворачивать в трей или выходить), частота обновления
статусов, задержка между запусками в профиле, экспорт и импорт всех
настроек в файл — удобно переносить на другой компьютер.

## Возможности по платформам

| Функция                     | Windows | macOS |
|-----------------------------|:-------:|:-----:|
| Запуск/остановка программ   | да      | да    |
| Отслеживание дерева процессов | да    | да    |
| Профили, заметки, оформление | да     | да    |
| Системный трей              | да      | да    |
| Автозапуск с системой       | да      | да    |
| Журнал событий, обновления  | да      | да    |
| Глобальные горячие клавиши  | да      | нет   |
| Менеджер автозагрузки       | да      | нет   |
| Распознавание .lnk / .url   | да      | нет   |

Часть функций пока только для Windows — на macOS они требуют особых
системных разрешений и будут добавлены позже.

## Обновления

Приложение само проверяет наличие новой версии при запуске и, если
она вышла, предлагает открыть страницу загрузки. Установка остаётся
за тобой — приложение только подсказывает.
